# TRP 模组更新工具

> 用于让客户端与服务器的模组同步，需要服主自己提供**模组列表文件**，当前目录的模版可以点击 5 次"可选"Tab 导出。

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
